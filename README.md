# Draft Gapped

Draft Gapped is a League of Legends pick/ban phase app.  
It emulates the pick/ban phase of a league of legends game in your web browser.  
You can play by yourself or with a friend/opponent. You can also chain multiple pick/ban phases.  
And there is a special gamemode where u can chain multiple pick/ban phases and each champion  
that is picked or banned will become unavailable in the next pick/ban phases.

## Features

- Enable/disable time countdown for pick/ban
- Create a lobby with 1-5 pick/ban phases
- Play with another player
- Save your drafts
